HTML Formм—җм„ң GET/POSTл§Ң м§Җмӣҗн•ҳлҠ” мқҙмң 

мү¬м–ҙк°ҖлҠ” л§ҲмқҢмңјлЎң лӢӨмқҢ мһҘмқ„ мң„н•ң мӮ¬м „ м§ҖмӢқмқ„ н•ҳлӮҳ мҢ“м•„ліҙмһҗ.

мҷң PUTкіј DELETEлҘј м§Җмӣҗн•ҳм§Җ м•ҠлҠ”к°Җм—җ лҢҖн•ң м •лҰ¬

  • мқјліём–ҙлӢӨ. кө¬кёҖ лІҲм—ӯкё°лҘј лҸҢл Өліҙмһҗ

PUT/DELETEлҘј Formм—җ л„Јмңјл ӨлҠ” мӢңлҸ„лҠ” мң„ кёҖм—җм„ң к°ҖмһҘ мһҗм„ёнһҲ лӮҳмҳӨкёҙ н–ҲлӢӨ.

(кө¬кёҖ лІҲм—ӯмқҙ нӣҢлҘӯн•ҳкёҙ н•ҙлҸ„) мқјліём–ҙлқј мӣҗліёмқ„ м•Ңм•„ліј мҲҳк°Җ м—ҶмңјлҜҖлЎң,
лҢҖмӢ  stackexchangeм—җ мҳ¬лқјмҳЁ кёҖлЎң м§§кІҢ м •лҰ¬н•ҙлҙӨлӢӨ.

  • мҙҲкё° HTML5 draftм—җлҠ” нҸ¬н•Ёлҗҳм–ҙ мһҲм—Ҳкі  Firefoxмқҳ лІ нғҖ лІ„м „м—җм„ңлҠ” мһ мӢң кө¬нҳ„н•ҳкё°лҸ„ н–Ҳм—ҲлӢӨ.
  • W3CлҠ” мқҙ л¬ём ңлҘј bug report 10671м—җм„ң л…јмқҳлҘј н–ҲлҠ”лҚ°,
  • Mike AmundsenмқҙлһҖ мӮ¬лһҢмқҙ м–ёкёүн•ҳкёё
    • лҰ¬мҶҢмҠӨлҘј мҲҳм •н•ҳкё° мң„н•ҙ PUTкіј DELETEлҘј мӢӨн–үн•ҳлҠ” кұҙ лӘЁлҚҳ лёҢлқјмҡ°м Җмқҳ XmlHttpRequest к°қмІҙлҘј мӮ¬мҡ©н•ҳл©ҙ к°„лӢЁн•ңлҚ°, мҠӨнҒ¬лҰҪнҠёлҘј мӮ¬мҡ©н•ҳм§Җ м•ҠлҠ” мғҒнҷ©м—җм„  мүҪм§Җ м•ҠлӢӨ.
    • мқҙ нҢЁн„ҙмқҖ л§ҺмқҖ мқјл°ҳм Ғмқё мӣ№ н”„л Ҳмһ„мӣҚ/лқјмқҙлёҢлҹ¬лҰ¬м—җм„ң кё°ліёмңјлЎң кө¬нҳ„н•ҳкі  мһҲлӢӨ.
    • (к·ёлҰ¬кі  мқҙлҘј мҡ°нҡҢн•ҳлҠ” л°©мӢқм—җ лҢҖн•ң л¬ём ңлҘј м–ёкёүн•Ё)
  • Tom WardropлһҖ мӮ¬лһҢлҸ„ нқҘлҜёлЎңмҡҙ кҙҖм җмқ„ м ңмӢңн•Ё
    • HTMLмқҖ HTTPмҷҖ л–јл Өм•ј л—„ мҲҳ м—ҶлҠ” кҙҖкі„лӢӨ. HTMLмқҖ мқёк°„мқ„ мң„н•ң HTTP мқён„°нҺҳмқҙмҠӨлӢӨ. л”°лқјм„ң мҷң HTTPм—җм„ң н•„мҡ”н•ң л©”мҶҢл“ңлҘј лӘЁл‘җ м ңкіөн•ҳм§Җ м•ҠлҠ” кІғмқёк°Җм—җ лҢҖн•ҙ мһҗлҸҷмңјлЎң мқҳл¬ёмқ„ к°Җм§Ҳ мҲҳ л°–м—җ м—ҶлӢӨ. кё°кі„лҠ” PUTкіј DELETEлҘј н•  мҲҳ мһҲмңјл©ҙм„ң мқёк°„мқҖ мҷң м•Ҳ лҗҳлҠ”к°Җ?
    • HTMLмқҙ мӢңл§ЁнӢұн•ң л§ҲнҒ¬ м—…мқ„ ліҙмһҘн•ҳлҠ” лҚ° л§ҺмқҖ мӢңк°„мқ„ н• м• н•ҳл©ҙм„ң, мӢңл§ЁнӢұн•ң HTTP мҡ”мІӯмқ„ ліҙмһҘн•ҳкё° мң„н•ң л…ёл Ҙмқ„ н•ҳм§Җ м•Ҡм•ҳлӢӨлҠ” кІғмқҖ лӘЁмҲңмқҙлӢӨ.
  • кІ°көӯ мқҙ лІ„к·ёлҠ” (HTML editorмқё) Ian Hicksonм—җ мқҳн•ҙ кі м№ҳм§Җ м•ҠлҠ” кІғмңјлЎң кІ°м •лӮ¬лӢӨ.
    • вҖңPUTмқ„ form л©”мҶҢл“ңлЎң мӮ¬мҡ©н•ңлӢӨлҠ” кІғмқҖ л§җлҸ„ м•Ҳ лҗңлӢӨ. л„Өк°Җ form payloadлҘј нҶөн•ҙ PUTмқ„ н•ҙм•јн•  мқјмқҖ м—Ҷмқ„ кІғмқҙлӢӨ. DELETEлҸ„ payloadк°Җ м—Ҷм–ҙм•јл§Ң л§җмқҙ лҗңлӢӨ. к·ёлҹ¬лӢҲ м—ӯмӢң formм—җм„ң м ңкіөн•  мқҙмң к°Җ м—ҶлӢӨ.вҖқ
  • н•ҳм§Җл§Ң нҸ¬кё°н•ҳм§Җ м•Ҡкі  м¶”к°Җ л“ңлһҳн”„нҠёк°Җ мҳ¬лқјмҷҖ мһҲлӢӨ
    • https://www.w3.org/TR/2015/NOTE-form-http-extensions-20150929/#http-put-form

мқјліём–ҙлЎң м •лҰ¬лҗң кёҖм—җм„ңлҸ„ мҷң PUTкіј DELETEк°Җ Formм—җм„ң м Ғн•©н•ҳм§Җ м•ҠмқҖм§ҖлҠ” мғқк°Ғн•ҳм§„ м•ҠмқҖ кІғ к°ҷлӢӨ.

м¶”к°Җлҗң л“ңлһҳн”„нҠёлҸ„ кІ°көӯ м• мҙҲмқҳ HTML Formмқҙ к°Җм§„ кё°лҠҘ/лӘ©м Ғмқ„ мҡ°нҡҢн•ҳм—¬ Formмқ„ нҷңмҡ©н•ң PUT/DELETEлҘј мӢӨн–үн•  мҲҳ мһҲлҸ„лЎқ Formмқҳ кё°лҠҘ нҷ•мһҘ(extension)мқ„ мҡ”кө¬н•ҳкі  мһҲлӢӨ.
м—¬кё°м—җм„ң DELETEлҠ” нҒ¬кІҢ мқҙм§Ҳк°җмқҙ м—Ҷм§Җл§Ң, PUTмқҳ кІҪмҡ° payloadлҘј нҶөн•ҙ н—ӨлҚ”лҘј кө¬м„ұн•ҳлҠ” л°©мӢқмқ„ м ңм•Ҳн•ҳкі  мһҲлӢӨ. к·ё л°©мӢқмқҖ кҪӨлӮҳ м–ҙмғүн•ҙм„ң кіјм—° HTML м—җл””н„°лӮҳ м»Өл®ӨлӢҲнӢ° м°Ёмӣҗм—җм„ң л°ӣм•„л“Өмқј мҲҳ мһҲмқ„м§ҖлҠ” лҜём§ҖмҲҳмқҙлӢӨ.

мқҙмҠҲлҘј лӢ«м•„лІ„лҰ° Ian Hickson мһ…мһҘм—җм„  Formм—җм„ң PUT/DELETEлҘј м§Җмӣҗн•ҳлҠ” кІҢ л„Ҳл¬ҙ лӢ№м—°н•ҳкІҢ л§җлҸ„ м•Ҳ лҗҳлҠ” мқјмқҙлқјкі  мғқк°Ғн–Ҳмқ„ кІғ к°ҷлӢӨ. мқҙмң лҠ”,

  • FormмқҖ м„ңлІ„м—җ м •ліҙлҘј м ңм¶ңн•ҳкё° мң„н•ҙ мЎҙмһ¬н•ңлӢӨ
  • GETкіј POSTлҠ” formм—җ мЎҙмһ¬н•ҳлҠ” лҢҖнҷ”нҳ• м»ЁнҠёлЎӨм—җ мһ…л Ҙлҗң к°’мқ„ ліҙлӮёлӢӨ
  • POSTмҷҖ PUTмқҳ к°ҖмһҘ нҒ° м°Ёмқҙ
    • RFC2616м—җм„ңлҠ” вҖҳмҡ”мІӯ URIмқҳ лӢӨлҘё мқҳлҜёвҖҷлқјлҠ” н‘ңнҳ„мқ„ мҚјкі 
    • RFC 7231м—җм„ңлҠ” вҖҳлҸҷлҙүлҗң representationм—җ лҢҖн•ң лӢӨлҘё мқҳлҸ„вҖҷлқјлҠ” н‘ңнҳ„мқ„ мҚјлӢӨ
    • POSTм—җм„ңмқҳ мҡ”мІӯ лҢҖмғҒ лҰ¬мҶҢмҠӨлҠ” лҸҷлҙүлҗң representationмқ„ вҖҳмІҳлҰ¬н•ҳлҠ”вҖҷ мЈјмІҙмқҙкі 
    • PUTм—җм„ңлҠ” лҸҷлҙүлҗң representationмқҖ лҢҖмғҒ лҰ¬мҶҢмҠӨмқҳ мғҒнғңлҘј лҢҖмІҙн•ңлӢӨ. мҰү, лҰ¬мҶҢмҠӨ мһҗмІҙк°Җ лҗҳм–ҙм•ј н•ңлӢӨ
  • л”°лқјм„ң
    • GETмқҖ formм—җ мһҲлҠ” м •ліҙлҘј мӨ„н…ҢлӢҲ лӮҳм—җкІҢ лҰ¬мҶҢмҠӨ(representation)лҘј мӨҳ
    • POSTлҠ” formм—җ мһҲлҠ” м •ліҙлҘј мӨ„н…ҢлӢҲ мқҙкұёлЎң мІҳлҰ¬ мўҖ н•ҙмӨҳ
  • лқјлҠ” мқҳлҜёк°Җ лҗҳм§Җл§Ң,
    • DELETEм—җм„ң URIлҠ” лҰ¬мҶҢмҠӨлҘј м •нҷ•н•ҳкІҢ мӢқлі„н•ҳкі  мҡ”мІӯмқ„ ліҙлӮҙкё° л•Ңл¬ём—җ formмқ„ нҶөн•ҙ к°’мқ„ м „лӢ¬н•  кІҢ м—Ҷмңјл©°
      • к¶Ңн•ңмқ„ нҡҚл“қн•ҳкё° мң„н•ҙм„  headerлҘј мқҙмҡ©н•ҙм•јн•  кІғ к°ҷлӢӨ
    • PUTлҸ„ м „мҶЎн•ҙм•јн•  кІғмқҖ лҢҖмғҒ лҰ¬мҶҢмҠӨлҘј лҢҖмІҙн•  representation к·ё мһҗмІҙмқёлҚ°, лҢҖмғҒ лҰ¬мҶҢмҠӨлҚ”лҹ¬ мЎ°к°ҒлӮң form лҚ°мқҙн„°лҘј мІҳлҰ¬н•ҳлқјкі  ліҙлӮј мҲҳлҠ” м—ҶлҠ” л…ёлҰҮ
